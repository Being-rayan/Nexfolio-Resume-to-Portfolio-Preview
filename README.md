# Nexfolio — Resume to Portfolio Generator

Nexfolio is an AI-assisted resume to portfolio generator that converts a user's resume PDF into a ready-to-use personal portfolio website.

The project allows users to upload a resume, extract important details, generate a portfolio layout, preview the website, and export the generated files for deployment on platforms like GitHub Pages or Vercel.

## Project Overview

Nexfolio was built to simplify the process of creating a personal portfolio website from a resume. Instead of manually writing HTML, CSS, and JavaScript for every user, the system reads the uploaded resume and generates a structured portfolio using extracted details such as name, role, skills, projects, experience, education, and contact information.

The project has also been tested using my friend's CVs, and it successfully generated portfolio websites that can be exported and deployed to create working public portfolio links.

## Key Features

- Resume PDF upload
- AI-based resume detail extraction
- Portfolio website generation
- Preview of generated portfolio
- Exportable HTML, CSS, and JavaScript files
- ZIP export support
- Deployable on GitHub Pages or Vercel
- Clean and responsive portfolio UI
- Tested with multiple resumes

## AI Disclaimer

Nexfolio uses AI to extract and organize resume information. Since AI-generated output may sometimes contain mistakes, users should always review and edit the generated portfolio before final deployment.
